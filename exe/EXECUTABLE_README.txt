# Society Management System - Executable Package

## Overview
This package contains the executable version of the Society Management System, a desktop application for managing residential society operations.

## System Requirements
- Windows 10 or higher (64-bit recommended)
- 4GB RAM minimum (8GB recommended)
- 100MB available disk space

## Installation
Simply extract this package to a folder of your choice. No additional installation is required.

## Running the Application
Double-click on `SocietyManagementSystem.exe` to start the application.

On first run:
- Username: `sysadmin`
- Password: `systemadmin`

It is highly recommended to change this password after the first login.

## Features
- User Authentication with role-based access control
- Resident Management
- Financial Tracking (Payments and Expenses)
- Bank Reconciliation (CSV and PDF support)
- Transaction Reversal
- Reporting (PDF generation)
- User Management
- Society Configuration
- Database Backup
- Dark Mode support

## User Roles
1. **System Admin**: Full access to all features
2. **Admin**: Manage residents, record payments/expenses, generate reports, manage users
3. **Treasurer**: Manage residents, record payments/expenses, generate reports
4. **Viewer**: View residents and financial reports
